Please ckeck the example behaviour as follows: 

CM0+	: LD1 will blink and then core will enter into DeepSleep mode (LD1 will off), press SW7 to wake-up core from DeepSleep (LD1 will on).
CM7_0	: LD2 will blink and then core will enter into DeepSleep mode (LD2 will off), press SW3 to wake-up core from DeepSleep (LD2 will on).
CM7_1	: Core will enter into DeepSleep mode.

Note	: All the cores should be in DeepSleep to check this functionality, please check after removing the debugger.
